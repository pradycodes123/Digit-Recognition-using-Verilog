
# Digit Recognition using Fixed-Point Neural Network in Verilog

This project implements handwritten digit recognition using a neural network trained in Python and simulated in Verilog using fixed-point arithmetic.

## Structure

python_model/
- Training code for MNIST neural network

data/mnist/
- MNIST dataset files

notes/PY-verilog/
- Learning notes and experiments

## Goal

- Train NN in Python
- Export weights
- Implement inference in Verilog
- Simulate hardware-level NN
